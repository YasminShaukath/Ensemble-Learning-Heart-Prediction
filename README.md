# Diabetes Prediction using Ensemble Learning

This project predicts whether a person has diabetes using Random Forest (an Ensemble Learning technique).

## Features
- Data preprocessing using Label Encoding
- Model comparison (Logistic Regression, Decision Tree, Random Forest)
- Random Forest selected as best model
- Streamlit web application for prediction

## Files
- 01.diabetes.ipynb → Model training
- 02.app.py → Streamlit application
- 03.diabetes_model.pkl → Saved model
- 04.diabetes.csv → Dataset
- 05.Screenshot.png → Output preview

## Run the App
streamlit run 02.app.py

## Result
The model predicts whether a person is diabetic or not based on input features.
